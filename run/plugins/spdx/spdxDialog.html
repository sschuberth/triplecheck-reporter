Create SPDX

Download a zipped file from the Internet and create an SPDX document from its contents. Only files with extension .zip and .tar.gz are supported at the moment. After pressing the "Create SPDX" button the new document can be open from the SPDX list.


Type the download URL (or use CTRL+C & CTRL+V to copy and paste)
