Link to folder on disk 

Click here to choose the folder where the source code files that were used for creating the SPDX document are located. This is the setting that links the document to real files on disk.

It is used when double-clicking on files from the treeview for viewing their content/source code.




 Experimental 

When you make changes on the source code, this option will create a new SPDX document and try to reuse as much as possible the information from the previous document.

Click this link to renew the document.
