File authorship 

Files at the beginning are marked as "unknown" in regards to its origin. Use the right-click menu to classify them. Instructions: wiki page.

It is possible to automatically verify which files are original or not using our internal database. Click the link below to launch an analysis.


 Action:  Verify originality


 Classification of files

 Original   %original%           3rd party   %3rdparty%

 Auto-generated   %automated%            Unknown   %unknown%

 Mod. 3rd party   %modded3rdparty%        Mod. auto-generated   %automixed% 
 


 Explanation 

  1. Original. Created by the developer(s) of this project
  2. 3rd party. Not created by the developer(s) of this project
  3. Auto-Generated. Files generated automatically by a tool, usually not eligible for claiming copyright or authorship
  4. Unknown. Files whose origin exists no certainty
  5. Mod. 3rd party. Files written by an outside developer and modified by the authors of this project
  6. Mod. auto-generated. Files generated automatically and later modified by the developer(s) of this project. Eligible for claiming copyright and authorship.
