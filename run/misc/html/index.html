FAQ
How is the score calculated?

The score ranks on a total of 10 points the quality of the free/open source licensing. It will give points to the number of files with declared license on their headers, point the copyright assignments and a check if it has documentation to help identify the usage and distribution conditions for a given piece of software. For a complete description of how this is done, please visit this wiki page.
